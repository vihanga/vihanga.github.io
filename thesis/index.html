Model-based Reinforcement Learning for Animating Virtual Characters

A novel approach to data-driven animation, portraying dynamic, flexible human-like behaviours

Abstract

Virtual characters enable immersive experiences in games and have significant potential for serious applications. However, creating believable character animation remains challenging - motion capture is expensive and inflexible, while existing data-driven methods lack the dynamic responsiveness needed for real-time interaction.

This thesis introduces RLAnimate, a model-based reinforcement learning framework that revolutionizes character animation. By using motion capture data as learning objectives rather than rewards, RLAnimate trains agents that can portray multiple human-like behaviors - waving, pointing, and conversational gestures - with unprecedented flexibility and quality.

The key innovation is treating animation as a model-based RL problem where agents learn latent dynamics of human movement. This enables real-time generation (5ms per frame) of animations that are statistically indistinguishable from human motion capture, as validated through rigorous perceptual studies with 28 participants.

RLAnimate represents the first data-driven methodology allowing dynamic, frame-by-frame portrayal of multiple human-like behaviors through a single agent with uniform input parameterization. The framework achieves this while requiring significantly less motion capture data than supervised learning approaches, making high-quality character animation accessible for interactive applications.
